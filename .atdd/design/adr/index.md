## ADR Creation Guide
1. 새 ADR 번호 할당 (순차적 증가)
2. 템플릿 복사하여 작성 (`.claude/skills/adr/adr-template.md`)
3. Status: Proposed로 시작
4. `/redteam` 실행하여 비평 수행
5. 비평 반영 후 Status: Accepted로 변경
